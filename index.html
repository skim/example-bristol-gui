About

A graphical user interface (GUI) for Bristol It is written in ANSI C, using GTK+. bristolgui is free software, licensed under the GPL v2.

Bristol is a synthesizer emulation package for Linux and BSD, emulating classic synthesizers.

In order to build bristolgui from source, you need to install it's dependencies.

  • GTK+2
  • lgui

Unpack the source code archive, open a terminal, change into the bristolgui source directory and run the following commands.

./autogen.sh
./configure
./make
sudo ./make install

For more information please refer to the included README file.
