About

Bristolgui is a graphical user interface (GUI) for Bristol, written in in ANSI C using the GTK+ library.

Bristolgui is free software, licensed under the GPL v2. It is developed and maintained by Martin Voigt, synthesizer images and descriptions provided by Nick Copeland.

Bristol is a synthesizer emulation package for Linux, written and maintained by Nick Copeland.

In order to build bristolgui from source, you need to install it's dependencies.

  • GTK+2
  • lgui

Unpack the source code archive, open a terminal, change into the bristolgui source directory and run the following commands.

./autogen.sh
./configure
./make
sudo ./make install

For more information please refer to the included README file.
